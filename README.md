# Описание проекта
## Идея
Проект будет представлять собой простую карточную игру, смесь пасьянса и покера.

## Описание
Игра будет приложением с графическим интерфейсом. У пользователя будет возможность создавать несколько профилей. Каждый профиль будет иметь свою статистику. Будет возможность просматривать статистику в виде графиков. Некоторые профили можно будет сделать _облачными_. У таких облачных профилей статистика будет храниться не локально, а удаленно, ее можно будет получить, войдя в игру с другого устройства.

### Библиотеки
- pygame
- matplotlib
- requests

### План
#### К 1 итерации планируется сделать
- Основная внутренняя логическая структура
- Графический интерфейс
- Несколько профилей

#### Ко 2 итерации планируется сделать
- Статистика профилей, графики
- Облачные профили
